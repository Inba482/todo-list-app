# TaskFlow - Elegant Task Management

This repository contains my internship project for **JavaScript Logic & State Management**, focused on building an interactive, state-driven To-Do List application using Vanilla JavaScript while mastering DOM manipulation, event handling, and client-side data persistence.

## 📌 Project Objective

Develop a fully functional client-side To-Do List application that demonstrates JavaScript fundamentals through dynamic user interactions, efficient state management, and persistent data storage using the browser's Local Storage API.

## ✨ Key Features

- Implemented complete CRUD (Create, Read, Update, Delete) functionality.
- Built dynamic DOM elements for real-time task management.
- Utilized event handling and event delegation for efficient interaction.
- Integrated `window.localStorage` to persist user data across browser reloads.
- Added task filtering for **All**, **Active**, and **Completed** states.
- Designed a responsive and intuitive user interface for seamless task management.

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- DOM API
- Local Storage API

## 🎯 Internship Learning Outcomes

Through this task, I gained practical experience in:

- Building interactive client-side web applications.
- Manipulating the DOM dynamically using JavaScript.
- Managing application state effectively.
- Implementing CRUD operations with clean JavaScript logic.
- Persisting user data using the Local Storage API.
- Developing responsive and user-friendly interfaces.

## 🚀 Expected Outcome

A fully functional, state-driven To-Do List application that enables users to create, manage, filter, and persist tasks, providing a smooth and interactive user experience across browser sessions.
